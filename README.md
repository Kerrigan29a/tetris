# Tetris
[![Build Status](https://travis-ci.org/Kerrigan29a/tetris.svg)](https://travis-ci.org/Kerrigan29a/tetris)

## Build

```bash
git clone https://github.com/Kerrigan29a/tetris.git
make
```

## Building options

- ENABLE_SCORE
- ENABLE_PREVIEW
- ENABLE_HIGH_SCORE
- ENABLE_UNICODE
- ENABLE_PRETTY_BOARD

## Keyboard Control

- j         - Left
- k         - Rotate
- l         - Right
- <SPACE>   - Drop
- p         - Pause
- q         - Quit
